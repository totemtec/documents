# Milvus 单机运行

官方文档
> https://milvus.io/docs/install_standalone-docker.md

### 下载脚本

```bash
wget https://raw.githubusercontent.com/milvus-io/milvus/master/scripts/standalone_embed.sh
```

### 启动

```bash
bash standalone_embed.sh start
```

### 关闭

```
bash standalone_embed.sh stop
```
