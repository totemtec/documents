# Windows Server 2016 初始化操作

### 关闭IE增强的安全配置方法

控制面板(Control Panal) -> 管理工具(Administrative Tools) -> 左侧选中 快速访问(Quick access) -> 服务器管理器(Server Manager) -> 本地服务器(Local Server) -> IE增强的安全配置(IE Enhanced Security Configuration) 启用(On)
修改为 管理员和用户都是设置为“关闭”



### 添加中文简体语言包
开始菜单 -> 设置(Settings) 齿轮图标 -> Windows Setting窗口
时间和语言(Time & language) -> 区域和语言 (Region & language)
区域选择中国(或者香港)，新增一个语言(Add a language)，搜索选择中文(中华人民共和国)即简体中文，完成

控制面板 -> 语言 -> 点击中文(中华人民共和国)的选项 -> 下载安装语言包，等待完成

控制面板 -> 语言 -> 点击中文(中华人民共和国)的选项 -> 使该语言成为主要语言，提示需要注销，注销后重新登录，系统显示就变为中文了
