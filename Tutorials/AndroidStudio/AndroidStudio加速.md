# AndroidStudio加速

- 忽略 Thumbs.db

~~~
File > Settings > Editor > File Types -> Ignored Files and Folders 添加一项 Thumbs.db
~~~

- 禁用未使用的插件

~~~
File > Settings > Plugins 取消勾选未用到的插件，比如Git
~~~

- 增大分配的内存

~~~
Preferences > Appearance & Behavior > System Settings > Memory Settings
~~~

- 修改 Gradle 使用国内镜像

见《Gradle使用国内镜像.md》
